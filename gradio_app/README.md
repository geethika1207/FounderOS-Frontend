# FounderOS

FounderOS is an AI-powered startup architect built with Gradio.

## Features

- User authentication
- AI startup idea analysis
- Database schema generation
- API endpoint suggestions
- MVP roadmap generation
- Startup risk analysis
- Chat with generated startup blueprint

## Backend

The frontend communicates with the deployed FounderOS FastAPI backend.
